First github page | first blog | first post

It's not all HTML and CSS

Well, I thought, it's time for my own blog. Primarily to document thoughts and findings for myself, but also for you, who are struggling with the same topics. Since I started reading articles in Medium (maybe later more about that), I stumbled over how to set up a personal homepage via github. And here it is! After trying out to setup a page with Jekyll (which was not as straight forward I thought it would be), after googling, I found themes from Start Bootstrap that could be used. With the well documented user guide I was able to create my first github page, my first blog, and my first post! One get's the hang of it just with a little knowledge about HTML, CSS and github.
